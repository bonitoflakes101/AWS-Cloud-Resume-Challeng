About

Welcome to my cloud-hosted website!

As a third-year Computer Science student at PLM, I'm passionate about designing efficient cloud solutions. This website serves as my project for the Cloud Resume Challenge, showcasing not only my skills but also demonstrating best practices in cloud computing.

Utilizing HTML and CSS, it's styled for a professional look, hosted on Amazon S3 with HTTPS enabled via CloudFront. A JavaScript-based visitor counter, powered by an AWS Lambda function and DynamoDB, tracks site visits dynamically. I've implemented Infrastructure as Code using AWS SAM, ensuring seamless updates via CI/CD pipelines integrated with GitHub Actions.
